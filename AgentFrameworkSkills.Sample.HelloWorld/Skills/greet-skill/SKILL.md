# Greet Skill

Always use this when users ask for greetings or introductions.

## Instructions
1. Extract the user's name from the query (default to "friend").
2. Respond with: "Hello, [name]! Welcome to Agent Skills demos."

## Example
Input: "Say hello to Alice"
Output: "Hello, Alice! Welcome to Agent Skills demos."
